Crea una landing page premium, moderna y orientada a conversión para “InmoAgentia”, una empresa que ofrece automatización inteligente de leads para agencias inmobiliarias mediante IA.

OBJETIVO DE LA WEB:
La landing debe transmitir confianza, tecnología, profesionalidad y facilidad de implementación. El objetivo principal es conseguir solicitudes de demo y propuestas personalizadas. El objetivo secundario es ofrecer una vía clara para que el visitante pueda contactar con un agente humano para resolver dudas sobre la aplicación, la instalación, la configuración o el funcionamiento del sistema.

DOMINIO:
inmoagentia.com

- Un único archivo `index.html`
- HTML + CSS + JavaScript vanilla
- Sin frameworks

- Solo se permite usar:

  - Font Awesome vía CDN
- Todo el CSS dentro de `<style>`
- Todo el JS dentro de `<script>` al final del documento
- El resultado debe ser completamente funcional al pegarse en Replit
- No devuelvas explicaciones: devuelve solo el código completo de `index.html`
